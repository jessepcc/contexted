# Code of Conduct

This project follows a simple standard: be respectful, specific, and constructive.

## Expected behavior

- Assume good intent, but speak plainly about technical concerns.
- Critique ideas, code, and decisions without attacking people.
- Be careful with intimate or sensitive topics. Contexted deals with personal writing, privacy, and trust.
- Keep bug reports, review comments, and product feedback actionable.

## Unacceptable behavior

- Harassment, intimidation, dogpiling, or personal attacks
- Publishing someone else's private data, tokens, logs, or memory excerpts
- Sexualized hostility, discriminatory language, or exclusionary behavior
- Repeated bad-faith argument, trolling, or derailing discussion

## Enforcement

If you need moderator help, do not post the incident publicly. Open a GitHub issue titled `Moderator contact requested` without details, and the maintainer will move the conversation to a private channel.

Project maintainers may edit, remove, or reject content that violates this code of conduct.
